Hospital Beds in United States

Data for 2015-2020

This visualisation is designed to demonstrate the number of hospital beds available in the United States. The data was retrieved from Kaggle by Igor Kiulian, Arthur Kiulian, Augaly S. Kiedi and Kristoffer Jan Zieba. The objective of this dataset is to demonstrate the hospital capacity during of each state. This visualisation is critical in understanding the reliability of each state in reacting to a high impact pandemic (such as COVID-19). The data retrieved is a combination from multiple sources. Created by: Hsien-Chi Angela Wu

Number of Beds Dot Density Map

The first visualisation is a dot density map, demonstrating the number of beds per 1,000 inhabitants for each state. A drop-down selection can be used to view each year's performance. The purpose of this visualisation is to show the comparison of each states performance in the number of beds provided. This is an important information, as it can easily address any potential flaws in the health system of USA. If a certain state lacks hospital beds it can impact the ability to react to a health crisis causing strain in the hospitals and can lead to an influx of patients without the necessary resource. This visualisation can also help understand the capacity of each state, for the health experts to appropriately assign and relocate patients. To view, please select year with the drop-down box.

Bed Type Usage Bar Chart

The second visualisation is a interactive bar chart showing each bed types available in the US. This demonstrates the popularity of each bed type, and can create a visual comparison of each type. Use the drop-down box to select the year to visualise. From the bar chart we can see that acute beds are the most common type of bed throughout all the years. This is not suprising as, the patients for acute beds are the more common type of cases.

Bed Type Definition
ACUTE Acute care beds are designed to treat patient with a brief but severe episode of illness, for conditions that are the result of disease or trauma, and during recovery from surgery.
ICU Also known as intensive care units, designed for patients requiring or potentially require advanced respiratory support.
PSYCHIATRIC Psychiatric beds are designed for accommodating patients with mental health problems.
OTHER This is for all other types of beds that do not directly fit into any of the bed types listed.
Each type serves a purpose for different situations, this visualisation is design to understand which bed type is in more desperate need for funding.

Top 20 States Bubble Plot

The last visualisation shows a bubble plot of the top 20 states with the highest beds per 1,000 inhabitants in every year. The visualisation includes multiple interactive features including, a drop-down box for the type of bed to view, zooming in and out to view the data and also mouse hover for highlighting and tooltip.

For each type of bed, it has been found that Iowa has the most acute beds in 2018, Montana has the most ICU and Psychiatric beds in 2018. This indicates that the states would be more prepared for an influx of patients, which is a vital data set in understanding crisis management for pandemics and flu season.
