Hospital Beds in United States

Data for 2015-2020

This visualisation is designed to demonstrate the number of hospital beds available in the United States. The data was retrieved from Kaggle by Igor Kiulian, Arthur Kiulian, Augaly S. Kiedi and Kristoffer Jan Zieba. The objective of this dataset is to demonstrate the hospital capacity during of each state. This visualisation is critical in understanding the reliability of each state in reacting to a high impact pandemic (such as COVID-19). The data retrieved is a combination from multiple sources.

Number of Beds Dot Density Map

The first visualisation is a dot density map, demonstrating the number of beds per 1,000 inhabitants for each state. A drop-down selection can be used to view each year's performance. The purpose of this visualisation is to show the comparison of each states performance in the number of beds provided. This is an important information, as it can easily address any potential flaws in the health system of USA. If a certain state lacks hospital beds it can impact the ability to react to a health crisis causing strain in the hospitals and can lead to an influx of patients without the necessary resource. This visualisation can also help understand the capacity of each state, for the health experts to appropriately assign and relocate patients.

Bed Type Usage Bar Chart

The second visualisation is a interactive bar chart showing each bed types available in the US. This demonstrates the popularity of each bed type, and can create a visual comparison of each type.
